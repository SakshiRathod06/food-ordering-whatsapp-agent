# AI Food Ordering WhatsApp Agent

## Project Overview

This project is an AI-powered WhatsApp Food Ordering Assistant developed using n8n.

The assistant helps customers:

* View food menus
* Place food orders
* Check inventory availability
* Track order status
* Get restaurant FAQs
* Receive food recommendations

## Technologies Used

* n8n
* OpenRouter AI
* WhatsApp Cloud API
* Google Sheets
* AI Agent Workflow

## Features

* WhatsApp-based food ordering
* Dynamic menu display
* Inventory checking
* Order confirmation generation
* FAQ support
* Customer-friendly responses

## Workflow

1. Customer sends a WhatsApp message.
2. AI Agent processes the request.
3. Inventory and FAQ data are retrieved from Google Sheets.
4. AI generates a response.
5. Reply is sent back through WhatsApp.
